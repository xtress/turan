Возможности

  • Один плеер для видео, аудио, фото и flash;
  • Более 100 базовых настроек и опций;
  • Результат за 5 минут в визуальном редакторе стилей;
  • Плеер написан на ActionScript 3, 9 версия Adobe Flash player и выше (более 95% пользователей Интернета по данным Adobe). Обновить свой Flash player можно здесь;
  • API для JavaScript;
  • Модули: RSS, Google Analytics и другие;
  • Интеграция с CMS;
  • Поддержка плейлистов;
  • Бесплатно, PRO, лицензия.
  • Нам доверяют уже более 50 000 пользователей!

Подробнее

Плеер воспроизводит все, что поддерживается технологией Flash:

Стандарт видео Flash video (H.263) MPEG-4 (H.264 — HD)
Форматы видео FLV F4V, MP4, MOV, M4A, MP4V, 3GP и 3G2
аудио — AAC (HE, LC)
Кодеки видео On2 VP6, Sorenson Spark H.264
Кодеки аудио MP3 (11, 22, 44 кГц)
Версия flash-плеера любая с 9.0.115.0

Основные возможности:

  • поддержка плейлистов различных конфигураций;
  • возможность скрывать прямые ссылки на файлы с помощью кодирования;
  • мультиязычный интерфейс;
  • возможность выбирать состав панели управления
    (более 20 кнопок и индикаторов можно установить в любой последовательности);
  • пользовательские подсказки;
  • полноэкранный режим для видео;
  • поддержка стримминга и псевдопоточной загрузки;
  • подключение веб-камеры;
  • горячие клавиши;
  • в интерфейсе нет нашей рекламы;
  • и еще много-много всего.

Возможности плейлистов:

  • расположение с любой стороны;
  • можно использовать картинку с подсказкой или просто текст;
  • можно задать размер элементов и цвета;
  • можно разбить плейлист на страницы.

Мы постарались сделать максимально гибким управление внешним видом и функционалом плеера, настройки позволяют:

  • задать любой размер и конфигурацию;
  • задать любой цвет (или несколько цветов) почти всем элементам интерфейса;
  • сделать внешний вид уникальным с помощью визуального редактора на нашем сайте;
  • гибко настроить внешний вид и функциональные возможности плейлистов;
  • установить картинку в качестве заставки, либо загружать определенный кадр для видео;
  • управлять плеером через JavaScript;
  • создавать скриншоты.
