flvS3Player - A plugin for FCKeditor

Overview

The flvS3Player plugin allows you to add and configure a small media player that playes flv.

Installation and Configuration

  1. Unzip the files into the plugins folder. ( fckeditor/editor/plugins/)
  2. Configure the editor to see the plugin. In your editor configuration file, add the line below:

    For Russian: FCKConfig.Plugins.Add( 'flvS3Player','ru') ;
    For English: FCKConfig.Plugins.Add( 'flvS3Player','en') ;

  3. Add the toolbar button to your preferred menu.

    FCKConfig.ToolbarSets[ "Default"] = [
    ['Source','DocProps','-','Save','NewPage','Preview','-','Templates','flvS3Player'],

  4. Save and upload your configuration file. If all goes well, you should see the flvS3Player icon on your toolbar.
